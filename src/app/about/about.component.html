О нас

Мы команда разработчиков из колледжа профессионального образования ПГНИУ.

Данный портал олимпиадного программирования был разработан для улучшения уровня навыков людей, которые хотят заниматься олимпиадным программированием.

Разработка была выполнена в рамках выполнения ВКР.

В качестве програмных решений были использованы:

  • ASP .NET Core в кчестве BACK-End части

  • FRONT-End на ANGULAR

  • GitHub Front

  • GitHub Back
